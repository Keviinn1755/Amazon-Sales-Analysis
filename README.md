# 📊 Amazon Sales Analysis Dashboard (Power BI)

## 🔍 Problem Statement
The objective of this project is to analyze Amazon sales data and track key business metrics to understand revenue trends, product performance, and customer feedback.

## 📈 KPI Requirements
- YTD Sales  
- QTD Sales  
- YTD Products Sold  
- YTD Reviews  

## 📊 Charts
- YTD Sales by Month  
- YTD Sales by Week  
- Sales by Category  
- Top 5 Products by Sales  
- Top 5 Products by Reviews  

## ⚙️ Features
- Data Cleaning  
- Data Modelling  
- DAX  
- Power Query  
- Time Intelligence  

## 🛠 Tools Used
- Power BI  
- Excel
